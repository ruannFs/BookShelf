# 📚 BookShelf

BookShelf é um aplicativo minimalista para catalogar livros e acompanhar sua leitura de forma simples, visual e organizada.

## 🎯 Justificativa do tema

Escolhi o tema **BookShelf** porque ele permite organizar livros de forma prática, visual e útil no dia a dia.  
Além disso, o projeto ajuda a demonstrar conceitos importantes do desenvolvimento Android, como **navegação entre telas**, **MVVM** e **persistência local de dados**.

---

## 🧩 Funcionamento do aplicativo

O aplicativo permite:

- cadastrar livros;
- editar informações existentes;
- remover livros;
- classificar por status:
  - **Lido**
  - **Lendo**
  - **Quero ler**
- visualizar uma lista organizada e minimalista;
- acompanhar um pequeno resumo da coleção no topo.

---

## ✨ Observações

- Interface desenvolvida com **Jetpack Compose**.
- Estrutura organizada seguindo o padrão **MVVM**.
- O design foi pensado para ser **leve, simples e elegante**.
- O aplicativo foi desenvolvido para facilitar a visualização e o controle da coleção de livros.
